# Task Hot State

Keep task-local scratch notes here when a task needs detailed implementation state that should not clutter `../../../docs/progress.md`.

This is not a second task template system and not a formal task card. Formal task cards still live under `../../../tasks/active/` and are created from `../../../templates/tasks/`.

## Naming

- prefer the same basename as the active task card
- example: `tasks/active/TASK-004-login.md` -> `TASK-004-login.md`

## Lifecycle

- keep the live note here while the task is active
- once the task is archived, move the note into `../../history/` or summarize it back into `../../../docs/progress.md`
