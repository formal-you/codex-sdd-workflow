# State

Use this folder for branch-aware and task-aware hot-state notes.

## Layout

- `hot/`
  - active scratch notes that may change frequently during implementation
- `history/`
  - retired hot-state notes that are still worth keeping after a task is archived

## Rules

- keep shared recovery summaries in `../docs/progress.md`
- keep long-lived project direction in `../docs/process.md`
- keep branch-local and task-local scratch state here so `progress.md` stays compact
