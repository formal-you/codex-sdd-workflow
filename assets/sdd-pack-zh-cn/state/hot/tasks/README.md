# Task Hot State（不是 task card）

当某个 task 需要详细实现便签，但这些内容不适合塞进 `../../../docs/progress.md` 时，请写在这里。

这里不是第二套 task 模板，也不是正式 task card。正式 task 仍然在 `../../../tasks/active/`，并且必须使用 `../../../templates/tasks/` 里的模板创建。

## 命名建议

- 优先与 active task card 同名
- 例如：`tasks/active/TASK-004-login.md` -> `TASK-004-login.md`

## 生命周期

- task 活跃期间保留在这里
- task 归档后，再按需移动到 `../../history/`，或压缩回 `../../../docs/progress.md` 的共享摘要
